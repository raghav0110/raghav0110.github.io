Mobile Systems Management

I am collaborating on a project which allows system administrators to monitor commercial data centers using mobile devices. We approach the problem by introducing a middleware between the mobile device and the monitored data centers. The middleware is used to process the streams of monitoring data from the data center target machines. It disseminates data to mobile devices when events are trigerred. I worked on developing the mobile application which receives alerts from the middleware and identified similar enterprise applications and identified ways to distinguish the novelty of our solution. I conducted evaluations on the middleware and the mobile device to measure there performance and compared them to current state of the art solutions. We have submitted a research paper to Middleware 2014 based on our work. During Fall 2014, I will continue to work on this project and start working on the middleware component of our framework.

Nature Nation

I lead a team to develop a mobile application that allows users to navigate to various wildlife species in the USA. We used the mobile application to take pictures of wildlife animals and sent the pictures and the GPS coordinates to a cloud service which identified the name and type of the animal. The mobile application synchronizes with the cloud service to receive a list of animals nearby. The cloud service is implemented such that it uses Amazon Mechanical Turk to identify the name and type of the animal. I worked on implementing the cloud service and the camera features within the mobile application.
