Controlling Network Costs in Multi-Site Cloud Deployments

I am currently assisting on a project that aims to control the network usage for distributed applications hosted in multiple cloud based datacenters. This work is a collaboration with multiple universities including TU Darmstadt, and Imperial College London.

My contributions included extending a pub-sub system, and researching related work on network traffic prediction. This involved understanding the theorotical concepts of different research papers. This work is being submitted to the USENIX Annual Technical Conference 2015.

(Note: This work has not been published, and the paper is linked only for reference in the Graduate School admission process)
